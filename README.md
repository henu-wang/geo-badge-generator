# GEO Score Badge Generator

Generate badges for your README to show your website's AI search readiness score.

## Usage

Add this badge to your README.md:

```markdown
[![GEO Score](https://img.shields.io/badge/GEO_Score-Check_Now-blue)](https://geoscoreai.com)
```

Result: [![GEO Score](https://img.shields.io/badge/GEO_Score-Check_Now-blue)](https://geoscoreai.com)

## How to Get Your Score

1. Visit [GEOScore](https://geoscoreai.com)
2. Enter your website URL
3. Get your AI search readiness score
4. Add the appropriate badge to your README

## What is GEO?

**Generative Engine Optimization (GEO)** is the practice of optimizing your website for AI-powered search engines like ChatGPT, Perplexity, Google AI Overviews, and Gemini.

Learn more: [What is GEO?](https://geoscoreai.com/blog/what-is-geo) | [GEO vs SEO](https://geoscoreai.com/blog/geo-vs-seo)
